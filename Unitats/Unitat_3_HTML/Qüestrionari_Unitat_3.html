Pregunta 1: Quin tipus de manteniment s'avança a l'error fent servir dades i sensors (SMART, sensors de temperatura) per substituir una peça just abans que falli?

  • a) Manteniment Preventiu
  • b) Manteniment Predictiu
  • c) Manteniment Correctiu
  • d) Manteniment Evolutiu
✅ Resposta correcta: b) Manteniment Predictiu


Pregunta 2: En l'àmbit de garanties i reparacions de components solts, què significa tramitar un procediment "RMA"?

  • a) Repair Machine Automatically (Reparar màquina automàticament)
  • b) Random Memory Access (Accés aleatori de memòria)
  • c) Return Merchandise Authorization (Autorització de devolució de mercaderia)
  • d) Remove Malicious Application (Eliminar aplicació maliciosa)
✅ Resposta correcta: c) Return Merchandise Authorization (Autorització de devolució de mercaderia)
